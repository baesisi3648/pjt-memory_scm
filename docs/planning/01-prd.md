# Memory SCM Intelligence Platform - PRD

## MVP 캡슐

1. 목표: 메모리 반도체 공급망 데이터를 통합 분석하여 이상 징후를 자동 감지하고, 드릴다운 분석을 제공하는 인텔리전스 플랫폼
2. 페르소나: 반도체 기업 SCM/구매팀 담당자, 반도체 애널리스트
3. 핵심 기능: FEAT-1 데이터 수집/통합 엔진, FEAT-2 공급망 이상 감지 & 알림, FEAT-3 드릴다운 분석 대시보드
4. 성공 지표 (노스스타): 주간 활성 사용자(WAU) 중 알림 확인 후 드릴다운 분석까지 완료한 비율
5. 입력 지표: 데이터 소스 수, 알림 정확도(오탐률)
6. 비기능 요구: 대시보드 로딩 3초 이내, 그래프 인터랙션 60fps
7. Out-of-scope: 보고서 생성/공유 (Phase 2), 가격/재고 카드 (후순위)
8. Top 리스크: 데이터 소스 확보 (공개 API 안정성, 유료 라이선스 비용)
9. 완화/실험: 공개 데이터로 MVP 검증 후 유료 소스 순차 추가
10. 다음 단계: 화면 명세 작성 (/screen-spec)

## 1. 제품 개요

- **서비스명**: Memory SCM Intelligence Platform
- **핵심 가치**: 분산된 메모리 반도체 공급망 데이터를 하나의 동적 그래프로 통합하여, 병목/이슈를 자동 감지하고 시각적으로 추적
- **타겟 사용자**:
  - 반도체 기업 SCM/구매팀: 공급망 현황 파악 및 구매 의사결정
  - 반도체 업계 애널리스트/리서처: 시장 분석 및 리포트 작성

## 2. 핵심 기능

### FEAT-1: 데이터 수집/통합 엔진
- **설명**: 다양한 소스(공개 API, 크롤링, 유료 라이선스)에서 메모리 반도체 공급망 데이터를 수집하고 정규화
- **Why**: 현재 데이터가 TrendForce, IC Insights 등 여러 곳에 분산되어 있어 교차 분석이 어려움
- **우선순위**: MVP 필수 (기반 인프라)

### FEAT-2: 공급망 이상 감지 & 알림
- **설명**: 가격/리드타임/재고 등의 비정상 변동을 자동 탐지하고, 밸류체인 그래프 위에 경고 아이콘으로 표시
- **Why**: SCM 담당자가 아침에 출근해서 가장 먼저 확인하는 것이 "어디에 문제가 있는가"
- **우선순위**: MVP 필수 (핵심 가치)

### FEAT-3: 드릴다운 분석 대시보드
- **설명**: 밸류체인 동적 그래프에서 노드(기업/클러스터) 클릭 시 사이드 패널로 상세 정보, 이슈 요약, 관련 뉴스, 업/다운스트림 관계를 탐색
- **Why**: 알림만으로는 부족, 원인을 추적하고 맥락을 파악해야 의사결정 가능
- **우선순위**: MVP 필수

### FEAT-4: 분석 보고서 생성 & 공유 (Phase 2)
- **설명**: 분석 결과를 팀/상사에게 보고할 수 있는 보고서 생성 및 공유 기능
- **우선순위**: MVP 제외, Phase 2

## 3. 사용자 스토리

- As a SCM 담당자, I want 공급망 이상 알림을 확인하고, so that 즉시 문제를 인지하고 대응할 수 있다
- As a SCM 담당자, I want 병목 지점을 그래프에서 클릭하여 원인을 분석하고, so that 데이터 기반 의사결정을 할 수 있다
- As a 애널리스트, I want 여러 데이터 소스를 통합된 뷰에서 교차 분석하고, so that 단일 소스에서 놓치는 인사이트를 발견할 수 있다
- As a 사용자, I want 관심 기업만 필터링하여 관계를 확인하고, so that 관련 없는 노이즈를 줄일 수 있다

| 여러 소스 교차 분석에 대한 니즈 | 가설 | 사용자 인터뷰 + MVP 사용 데이터 |

## 9. 실패 대응

- **MVP 정의**: 밸류체인 그래프 + 병목 감지 + 드릴다운 최소 기능
- **실패 시나리오**: 공개 데이터만으로 유의미한 알림 불가
- **학습 목표**: 어떤 데이터 소스 조합이 가장 유의미한 알림을 생성하는지 파악
- **Pivot 옵션**: 데이터 시각화 컨설팅 도구로 전환
